Musiquence Finance

Musiquence Finance is a web-based, gamified platform specifically designed for the cognitive rehabilitation of financial literacy in elderly post-stroke patients. By simulating real-world economic transactions, the application helps patients regain their autonomy and financial independence in a safe, controlled, and engaging environment.  

🎯 Key Features

The platform operates as a complete rehabilitation ecosystem, divided into several core modules:  

User Authentication & Profile Management: A simplified registration and login process designed to minimize cognitive load. Users can personalize their profiles with a real photograph to foster a sense of identity within the app.  

Intuitive Activity Hub: A main menu featuring large, clear, and interactive visual cards for seamless navigation between tasks.  

Statistics & Evolution Dashboard: Converts gameplay data into objective clinical indicators. It uses line and bar graphs to track patient progress over time, with filters available by specific game and difficulty level.

🎮 The 6 Simulation Mini-Games

Each game features 3 difficulty levels (Easy, Medium, Hard) to adapt to the patient's cognitive recovery stage:  

Math Gym (Ginásio das Contas): Simulates a notebook to train mental calculation, executive functions, and working memory through randomized arithmetic operations.

Identify Coins/Notes (Identificar Moedas/Notas): Uses realistic Euro currency representations to train visual recognition, value addition, and advanced payment simulations.  

Supermarket (O Supermercado): Integrates reading and applied math by having users fetch items from a shopping list on virtual shelves and validate the final receipt.  

Pharmacy (A Farmácia): Focuses on health literacy and precision. Users must identify correct medications (often with similar packaging) and calculate the exact change.  

ATM (O Multibanco): Replicates an ATM interface to practice memory and motor coordination through PIN entries, cash withdrawals, service payments, and transfers (MB WAY and IBAN).  

Online Shopping (As Compras Online): Familiarizes seniors with e-commerce. It challenges users with budget management, shipping costs calculation, and digital payment methods like Credit Cards, MB WAY, and ATM references.  

🛠️ Technology Stack

The project was developed prioritizing speed, lightweight execution, and ease of maintenance:  

Frontend: HTML5, CSS3, and Vanilla JavaScript.  

Backend: Node.js (entry point via server.js).  

Data Storage: JSON files for instantaneous data read/write operations without server delays.  Data Visualization: Chart.js library for dynamic evolution graphics.
